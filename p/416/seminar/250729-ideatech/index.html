脱・トップセールス依存！チーム全体で大手・エンプラ案件を獲る仕組みづくり

開催終了

営業組織の成果が伸び悩み、どうすれば解決できるかわからないと課題を抱えている方が多くいます。

  • エースがいないと大手・エンプラ案件が取れない、トップセールス依存から抜け出せない
  • メンバーによって提案品質にバラツキがあり、安定した成果が出せない
  • デモ後の反応はいいのに受注に繋がらない、失注率の高さに悩んでいる

このようなお悩みをお持ちの方は、ぜひ本セミナーにご参加ください。

本セミナーでは、かつて同じような課題に直面しながらも属人化を解消し、チーム全体で安定して
大手・エンプラ案件を獲得できるようになった再現性のある仕組みづくりを現場目線で解説します。

セミナー概要

Session 1｜本セミナーのためだけの完全独自調査レポートをもとにパネルディスカッション
Session 2｜新人が１か月でトップセールスの売上を2倍に！営業組織に再現性をもたらす話術AIとは？（Umee Technologies株式会社）
Session 3｜デモ後失注を防ぐ『体験継続』の仕掛けづくり（PLAINER株式会社）
Session 4｜『なぜ今、このサービスが必要なのか』を担当者・決裁者に納得させるコンテンツ戦略とは？
（株式会社IDEATECH）

登壇者紹介

中川 恵介 氏　Umee Technologies 株式会社　マーケティング/カスタマーサクセス

新卒で人材系メガベンチャーへ入社。営業、マーケティングを経験後、社内新規事業の立ち上げに携わる。
その後ソフトバンク子会社、株式会社LegalOn Technologies（旧：株式会社LegalForce）で
マーケティングを経験後、現職。

阿久津　卓哉 氏　PLAINER株式会社　事業開発

新卒で株式会社東日本銀行に入行し、企業開拓専担者として法人新規開拓や融資提案営業に従事。
その後株式会社favyにてデジタルマーケティングやコンサルティング、新規事業開発に携わる。
株式会社Rocketsでは、セールスエンゲージメントSaaS「LEADPAD」をローンチして、グロースを経験。
執行役員COOとしてビジネスサイドを統括。2025年4月にPLAINERにジョイン。
ウェビナーやイベントの登壇も多数。

競 仁志　株式会社IDEATECH　取締役

2013年株式会社ネットプロテクションズへ入社。その後自身で会社を創業。
海外企業の日本市場進出支援を行う。2020年より株式会社IDEATECH（旧：ベイニッチ）の取締役へと就任。リサーチデータマーケティング「リサピー®︎」事業の立ち上げ・推進の責任者となる。
企画立案数は3,000以上、累計担当企業社数は300社以上を実施。
BtoB領域のPR・マーケティング・セールスを「コンテンツ」で統合し、売上向上を実現。
